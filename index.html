v0.2
Input: CAN ID: form with binary string, e.g. [ 100101 ]
Input: Payload: form with binary string, e.g. [ 00100111001001 ]

Output:
frame bits min, max
whole frame without stuff bits as binary string, e.g. 100101001001...
while frame without stuff bits broken down per parts
- SOF: 1
- ID: 1001001
- etc.
whole frame WITH stuff bits as binary string, e.g. 100101001001... (optional, make them bold?)
